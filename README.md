## Description 
>lifeLvling is a web-based platform made to support those who wish to get themselves into shape but lack the motivation and time, Allowing customers to get into shape with the motivation of gaining online points, reaching higher levels that is displayed in the course leaderboard. Customers will be able to search the app for workout courses provided by fitness instructors that most fit the workouts/routines that they've been searching for, view instructor and course ratings, enroll/unenroll in courses, and leave their own reviews. Instructors create and manage their courses, providing routines and workout due dates to students, grant students points, and can view student progression in the course. The system supports two user roles, customer and provider, each with their own privacies and permissions.

## App Functions
1. Customer (the user with the customer role):
    1. Create/modify customer profile - User will be required to make a profile, or sign into an existing profile
    2. View available courses - customers will be brought to a dashboard, showing popular/available courses, a search bar will allow for specific courses to be searched for.
    3. enroll/drop available courses - When a course is found and has open space available, the customer can enroll. When veiwing that same course when already enrolled, a drop course option will be available.
    4. Write reviews for couurses and course instructors - When viewing a course, a review option will be available where the customer can see reviews of other customers. A customer must be enrolled, or previously had been enrolled in the course to write a review.
2. Provider (the user with the provider role):
    1. Create/modify/remove provider profile - Providers register an instructor account and manage their bio, certifications, and profile info through a provider settings page. They can update details or delete their account from the dashboard.
    2. Create courses - Providers can create courses using a course creation form where they add workout plans, schedules, difficulty level, due dates, and point rewards. Courses are saved and listed in the searchable course catalog.
    3. View student/overall course statistics -  Providers have a dashboard that shows enrollment counts, student progress, completion rates, and average course ratings using tables and progress indicators.
    4. Reply to reviews - Providers can respond directly to customer reviews under each course through a reply button, allowing follow-up feedback or clarification.
